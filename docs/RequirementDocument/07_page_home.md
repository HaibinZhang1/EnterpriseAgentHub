# 8.1 首页

## 页面目标
作为系统入口，集中展示当前用户最关心的待办、动态和推荐内容。

> 阶段说明：P1 首页只展示使用闭环相关信息；发布、审核、管理类待办随 P2 企业治理能力启用。

## 核心模块
- 我的待办
- 我的安装情况
- 我的发布情况（P2）
- 热门 / 推荐 skill
- 最近更新
- 通知摘要

## 展示内容

### 我的待办
- 可更新 skill 数量
- 待审核数量（P2）
- 待处理发布数量（P2）
- 被退回修改数量（P2）

### 快捷入口
- 查看我的 skill
- 查看工具管理
- 查看项目管理
- 发布 skill（P2）
- 进入审核（P2）

### 推荐内容
- 热门 skill
- 最新上架 skill
- 最近更新 skill
