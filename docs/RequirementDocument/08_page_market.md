# 8.2 市场

## 页面目标
为用户提供 skill 的发现、搜索、筛选、排序、安装与详情查看能力。

> 阶段说明：市场页属于 P1 Desktop 核心范围。P1 支持基础搜索、筛选、排序、详情查看、安装/更新入口；分类/标签在 P1 仅展示服务端已有字段，不负责定义正式分类体系。复杂推荐算法、排行榜权重调优和分类标签治理进入 P2。

## 页面结构
- 搜索框
- 筛选区
- 排序区
- skill 列表
- 排行榜入口

## 搜索能力
支持按以下字段搜索：
- skill 名称
- 描述
- 标签
- 作者
- 部门
- skillID

## 筛选能力
支持按以下维度筛选：
- 部门
- 分类
- 工具兼容性
- 风险等级
- 发布时间
- 更新时间
- 是否已安装
- 是否已启用
- 是否仅显示我有权限查看的 skill
- 是否包含摘要公开 / 详情公开 / 全员可安装 skill

> P1 可先实现部门、工具兼容性、是否已安装/已启用、权限可见性等高价值筛选；分类、风险等级等依赖治理字段的筛选可随 P2 字段固化后增强。

## 排序能力
支持以下排序方式：
- 综合排序
- 最新发布
- 最近更新
- 下载量
- Star 数
- 搜索相关度

---

## Skill 卡片
卡片建议展示：
- 图标
- 显示名称
- skillID（次级展示）
- 简短描述
- 标签 / 分类
- 作者
- 作者部门
- 当前版本
- Star 数
- 下载量
- 风险等级标签
- 最后更新时间
- 当前状态：未安装 / 已安装 / 已启用 / 有更新

### 卡片按钮
- 安装
- 更新
- 已安装
- 已启用
- 进入详情

---

## 市场详情页
完整详情页需展示：

### 基础信息
- skillID
- 显示名称
- 图标
- 描述
- 标签 / 分类
- 作者
- 所属部门
- 维护者
- 联系方式（如有）
- 当前版本
- 发布时间
- 更新时间

### 使用信息
- 适用工具
- 适用项目
- 支持环境说明
- 安装说明
- 启用说明
- 依赖项（如有，仅展示，不做自动解析）

### 内容信息
- README / 使用说明
- 截图 / 演示资料（如有）
- 文件在线预览（支持 md、txt、xml、html、json 等简单文本）
- 不支持预览的文件提示下载

### 审核与安全信息
- 审核通过时间
- 审核人
- 风险等级
- 审核意见摘要
- 系统初审结果
- 风险说明

### 版本信息
- 当前版本
- 历史版本列表
- 每个版本的变更说明
- 版本状态（已发布、已归档等）

### 操作区
- 安装
- 下载 zip
- Star / 取消 Star
- 启用到工具
- 启用到项目
- 查看已启用位置

---

## 受限详情页
当用户属于"摘要公开但无详情权限"时，进入受限详情页，仅展示：
- skill 名称
- 发布人
- 部门
- Star 数
- 下载量
- 分类 / 图标（如有）
- "该 skill 暂未向你开放详情"提示

可预留：
- 申请访问
- 申请使用

---

## 排行榜
排行榜拆分为两类：

> 阶段说明：P1 可展示服务端直接返回的简单榜单；综合权重、最近增长榜、榜单运营规则进入 P2/P3。

### 权限内排行榜
仅展示用户当前可查看详情的 skill，支持：
- 全部榜
- 部门榜
- Star 榜
- 下载榜
- 最近增长榜（预留）

### 全站曝光榜
仅展示满足公开条件的 skill，默认展示前 20 名：
- 摘要公开：展示摘要字段
- 详情公开：展示详情入口
- 全员可安装：展示详情与安装入口
