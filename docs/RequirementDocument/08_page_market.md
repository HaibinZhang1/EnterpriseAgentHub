# 8.2 社区（原市场）

## 页面目标
社区承担 Skill 的发现、搜索、筛选、排序、安装与详情查看能力。

当前正式交付中，“市场”页面已经按新的 IA 收口为一级入口 **社区**。

> 说明：社区页属于当前版本 Desktop 核心范围。当前版本支持搜索、排序、标签筛选、卡片网格、热榜侧栏、安装/更新入口与发布按钮；MCP / 插件仅保留占位态。

## 页面结构
- 左侧社区入口：`Skill / MCP / 插件`
- 顶部统一功能栏
- 搜索与排序区
- 标签按钮行
- Skill 卡片网格
- 右侧热榜侧栏
- 右上角 `发布 Skill` 主按钮

## 搜索能力
支持按以下字段搜索：
- skill 名称
- 描述
- 标签
- 作者
- 部门
- skillID

## 筛选能力
支持按以下维度筛选：
- 分类 / 标签
- 排序
- 其他高阶筛选维度仍由服务端字段逐步增强

当前正式页面优先交付：
- 搜索词
- 标签行
- 排序

## Skill 卡片

社区当前采用卡片网格，而不是单列大摘要列表。

每张卡片展示：
- 首字母图标
- 显示名称
- 简短描述
- 分类标签
- 当前状态
- 作者
- 作者部门
- 当前版本
- Star 数
- 下载量

### 卡片操作
- 主操作：安装 / 更新 / 管理范围 / 启用范围
- 次操作：详情
- 收藏 / 取消收藏

## Skill 详情

当前正式交付的 Skill 详情优先通过覆盖层展示，而非独立详情页。

完整详情展示：

### 基础信息
- skillID
- 显示名称
- 描述
- 作者
- 所属部门
- 当前版本
- 本地版本
- 更新时间

### 使用与兼容信息
- 适用工具
- 支持环境说明

### 内容信息
- README / 使用说明摘要

### 审核与安全信息
- 风险等级
- 审核意见摘要
- 风险说明

### 操作区
- 安装
- 更新
- 收藏 / 取消收藏
- 启用范围
- 查看已启用位置

## 受限详情
当用户属于“摘要公开但无详情权限”时，仅展示：
- skill 名称
- 发布人
- 部门
- Star 数
- 下载量
- 分类 / 图标（如有）
- “该 skill 暂未向你开放详情”提示

## 热榜侧栏

社区右侧保留独立热榜侧栏，不再回到主页。

当前版本默认展示简单热榜能力：
- 下载榜
- Star 榜
- 榜单行点击后进入 Skill 详情覆盖层

榜单行字段：
- 首字母图标
- Skill 名称
- 单个分类标签
- Star
- 下载量

## 发布入口

发布中心不再作为左侧入口或一级导航。

当前社区通过顶部功能栏右侧主按钮提供：
- `发布 Skill`

行为规则：
- 已登录：打开发布中心覆盖层
- 未登录：先打开登录模态，再进入发布流程
