# 8.4 审核

## 页面目标
为审核员提供统一的待办查看入口、快速摘要区和全屏审核详情页。

> 说明：审核页已进入 Desktop 正式导航，并对**在线且有菜单权限的管理员**开放处理入口。审核工作台本身采用“队列 + 快速摘要”，用于快速判断和分流；锁单、通过初审、同意、退回、拒绝、提交包文件预览/下载和完整历史时间线统一在全屏审核详情页中完成。

## 页面结构
- 待审核
- 审核中
- 已审核
- 快速摘要
- 全屏审核详情
- Skill 详情（辅助查看，不承载审核动作）
- 审核历史时间线

## 页签角标
当存在待审核项时，"审核"页签需显示待审核数量。

## 工作台快速摘要

工作台右侧保留快速摘要区，用于在不离开队列的情况下判断审核单优先级。快速摘要不得承载最终审核动作。

### 展示内容
- skill 基础信息
- 当前状态、风险等级、单据类型
- 提交人、提交部门、提交时间
- 当前版本 / 目标版本，或权限变更前后摘要
- 公开级别与授权范围摘要
- 系统预检查异常数量与首个异常摘要
- 最近一条审核历史

### 可执行动作
- 打开审核详情
- 打开 Skill 详情

> 约束：同意、退回修改、拒绝、通过初审等最终审核动作只能在全屏审核详情页执行，避免审核员在信息不完整时误操作。

---

## 审核 SLA 与异常分支

### SLA
- 锁单超时：5 分钟
- 超时自动释放，生命周期恢复到领取前状态（待人工复核或待管理员审核），并写入 `lock_expired` 审核历史
- 不提供强制解锁
- 同级审核员均可见、谁先领取谁处理

### 异常分支
- 系统初审失败：**不直接拒绝**，进入"待人工复核"
- 人工复核后可：
  - 同意进入正式审核
  - 退回修改
  - 审核拒绝
- 阻塞预检查失败包括 `SKILL.md` 缺失、SemVer / 版本递增、包大小、文件数、公开级别或授权范围等项目；审核员可风险覆盖，但必须填写审核意见，系统将该意见记录为 `override_precheck` 覆盖理由
- `通过初审` 与 `同意` 使用同一套阻塞预检查覆盖规则：无审核意见时拒绝，有审核意见时允许合法且已领取的审核员覆盖
- 不考虑审核通过后再触发高风险追溯复核
- 不考虑审核员离线 / 离职时的自动接管

## 审核详情

审核详情采用全屏 overlay，复用 Skill 详情页的视觉骨架，但按照审核员决策顺序组织内容。

### 展示内容
- skill 基础信息
- 授权范围
- 公开级别
- 文件在线预览 / 下载
- 系统预检查结果
- 分类与标签
- 功能描述
- 风险说明
- 历史版本
- 发布者信息
- 审核历史
- 权限变更记录（如有）

### 页面分区
- 顶部概览：skill 名称、skillID、描述、状态、风险、单据类型、提交人、部门、提交时间、当前审核人
- 本次变更：首次发布展示目标版本、公开级别和授权范围；更新发布展示当前版本到目标版本；权限变更展示当前权限到申请权限
- 审核动作：展示审核意见输入框，并按 `availableActions` 展示开始审核、通过初审、同意、退回修改、拒绝
- 系统预检查：警告项需要明确展示，通过项可弱化展示；存在阻塞型警告时，在审核动作区提示“填写审核意见后可风险覆盖”
- 包内容预览：复用提交包文件预览能力
- Skill 信息：展示分类、标签、描述、版本与风险信息
- 审核历史：按时间线展示历史动作、操作人、说明和时间

### 文件预览规则
- 支持 `.md` / `.markdown` / `.txt` 在线预览
- 其他文件仅提供下载，并提示"不支持预览"
- 默认优先选中 `SKILL.md`；不存在时选第一个可预览文件
- 单文件预览内容超过 256 KB 时显示"内容已截断"

### 可执行动作
- 切换待审核 / 审核中 / 已审核视图
- 查看快速摘要
- 打开审核详情
- 查看历史审核意见
- 开始审核（锁单）
- 通过初审（人工复核 -> 待管理员审核）
- 同意
- 拒绝
- 退回修改

### 拒绝 / 退回说明
- 支持选择原因模板
- 支持补充自定义说明
