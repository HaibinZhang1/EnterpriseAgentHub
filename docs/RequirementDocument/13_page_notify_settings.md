# 8.8 通知

## 页面目标
统一展示与用户相关的系统消息和流程通知。

> **说明：** 当前版本实现**应用内消息中心**与本机通知缓存；企业 IM、系统托盘通知、通知保留期限和清理策略暂不展开。

## 通知类型
- skill 有新版本可更新
- skill 权限已收缩
- 当前安装副本不可再更新
- 连接状态异常 / 恢复
- 工具或项目路径异常
- skill 已进入审核
- skill 审核通过
- skill 审核拒绝
- skill 被退回修改
- skill 已发布
- skill 已下架
- skill 已重新上架
- skill 已归档
- 你有新的待审核任务

## 通知字段
- 通知标题
- 通知摘要
- 关联对象
- 时间
- 已读 / 未读状态

## 操作
- 标记已读
- 全部已读
- 跳转详情页

# 8.9 设置

## 功能范围
- 多语言设置
- 默认语言自动识别
- 基础偏好设置

## 多语言规则
- 系统支持多语言
- 默认根据用户地区自动选择语言
- 用户可手动切换
