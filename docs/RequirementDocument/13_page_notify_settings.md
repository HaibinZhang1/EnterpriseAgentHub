# 8.8 通知

## 入口与目标
通知不再作为一级页面存在，统一收敛为顶栏右上角铃铛按钮 + 通知面板。

## 面板结构
- 标题：通知
- 全部 / 未读 切换
- 全部已读
- 最近 20 条通知列表

## 当前版本保留的通知类型
- 审核进度相关
  - skill 已进入审核
  - skill 审核通过
  - skill 审核拒绝
  - skill 被退回修改
  - 你有新的待审核任务
- Skill 更新
  - 某个已安装 Skill 有新版本可更新
- 软件更新
  - 桌面客户端本身有新版本

## 不再展示的内容
- 独立通知中心页
- 搜索、删除、独立“查看全部页面”
- 长期归档体系

## 通知项字段
- 通知标题
- 通知摘要
- 图标
- 时间
- 已读 / 未读状态

## 点击行为
- 新的待审核任务 / 管理员审核任务 -> 跳转 `管理 > 审核`
- 作者视角审核进度 -> 打开“发布中心覆盖层 > 我发布的”
- Skill 更新 -> 跳转 `本地 > Skills` 并优先切到“待更新”视图
- 软件更新 -> 打开轻量更新提示弹窗

## 面板操作
- 全部已读
- 点击单条通知后关闭面板
- 点击面板外区域关闭
- 再次点击铃铛关闭

# 8.9 设置

## 功能范围
设置采用弹窗形态，不作为一级页面存在。弹窗结构为左侧设置列表、右侧详情，默认只展示当前分组的关键项，避免在单屏内堆叠过多信息。

当前可列入设置的项目：

- 常规偏好
  - 多语言设置
  - 默认语言自动识别
  - 主题设置
- Agent 接入
  - 模型服务选择
  - API Key 设置
  - Base URL 设置
  - 默认模型设置
- 本地环境
  - Central Store 路径展示
  - 企业服务地址展示
  - 当前连接状态展示
- 同步与更新
  - 是否显示安装 / 更新结果
  - 联网后是否同步本地事件
  - 刷新启动上下文
  - 客户端版本与可用更新提示

不放入设置、仍保留在原页面的项目：

- 工具路径、项目路径和启用范围仍属于 `本地` 页。
- Skill 安装、更新、卸载、启用和目标绑定仍属于 `社区` / `本地` 主流程。
- 发布、审核、部门、用户和 Skill 治理仍属于 `发布中心` 或 `管理` 入口。

## Agent API Key 规则
- API Key 设置用于后续接入 Agent 调用链路。
- 当前阶段可先作为本机偏好保存，并在界面上明确提示后续需要迁移到更安全的系统密钥存储。
- API Key 输入框默认按密码字段展示，不在列表摘要中明文展示。
- 未配置 API Key 时，主页 Agent 探索仍可作为入口展示，但真实模型调用需提示用户先完成配置。
- 支持 OpenAI、Anthropic 和自定义兼容服务的基础配置口径，避免后续只绑定单一模型供应商。

## 打开方式
- 设置不再作为一级页面存在
- 统一通过右上角头像菜单打开设置模态

## 多语言规则
- 系统支持多语言
- 默认根据用户地区自动选择语言
- 用户可手动切换
