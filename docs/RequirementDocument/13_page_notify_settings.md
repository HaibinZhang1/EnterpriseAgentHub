# 8.8 通知

## 页面目标
统一展示与用户相关的系统消息和流程通知。

> **说明：** P1 仅实现**应用内消息中心**。企业 IM 集成、系统托盘通知、通知保留期限和清理策略为 P3 功能。
> 阶段说明：P1 交付应用内消息中心和本机状态通知；发布/审核相关通知随 P2 发布审核流程启用；企业 IM、系统托盘、通知生命周期策略进入 P3。

## 通知类型
- skill 有新版本可更新
- skill 权限已收缩
- 当前安装副本不可再更新
- 连接状态异常 / 恢复
- 工具或项目路径异常
- skill 已进入审核（P2）
- skill 审核通过（P2）
- skill 审核拒绝（P2）
- skill 被退回修改（P2）
- skill 已发布（P2）
- skill 已下架（P2）
- 你有新的待审核任务（P2）

## 通知字段
- 通知标题
- 通知摘要
- 关联对象
- 时间
- 已读 / 未读状态

## 操作
- 标记已读
- 全部已读
- 跳转详情页
