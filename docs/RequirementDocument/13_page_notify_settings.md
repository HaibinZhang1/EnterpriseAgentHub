# 8.8 通知

## 入口与目标
通知不再作为一级页面存在，统一收敛为顶栏右上角铃铛按钮 + 通知面板。

## 面板结构
- 标题：通知
- 全部 / 未读 切换
- 全部已读
- 最近 20 条通知列表

## 当前版本保留的通知类型
- 审核进度相关
  - skill 已进入审核
  - skill 审核通过
  - skill 审核拒绝
  - skill 被退回修改
  - 你有新的待审核任务
- Skill 更新
  - 某个已安装 Skill 有新版本可更新
- 软件更新
  - 桌面客户端本身有新版本

## 不再展示的内容
- 独立通知中心页
- 搜索、删除、独立“查看全部页面”
- 长期归档体系

## 通知项字段
- 通知标题
- 通知摘要
- 图标
- 时间
- 已读 / 未读状态

## 点击行为
- 新的待审核任务 / 管理员审核任务 -> 跳转 `管理 > 审核`
- 作者视角审核进度 -> 打开“发布中心覆盖层 > 我发布的”
- Skill 更新 -> 跳转 `本地 > Skills` 并优先切到“待更新”视图
- 软件更新 -> 打开轻量更新提示弹窗

## 面板操作
- 全部已读
- 点击单条通知后关闭面板
- 点击面板外区域关闭
- 再次点击铃铛关闭

# 8.9 设置

## 功能范围
- 多语言设置
- 默认语言自动识别
- 主题设置
- Central Store 路径展示
- 基础偏好设置
- 是否显示安装 / 更新结果
- 联网后是否同步本地事件
- 刷新启动上下文

## 打开方式
- 设置不再作为一级页面存在
- 统一通过右上角头像菜单打开设置模态

## 多语言规则
- 系统支持多语言
- 默认根据用户地区自动选择语言
- 用户可手动切换
