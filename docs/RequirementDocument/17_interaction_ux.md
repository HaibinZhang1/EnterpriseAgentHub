# 12. 交互与状态提示要求

## 12.1 空状态
以下页面需要提供明确空状态提示：
- 社区无结果
- 本地 Skill 列表为空
- 我发布的为空
- 待审核为空
- 工具未检测到
- 项目为空
- 通知为空

## 12.2 错误状态
以下场景需提供明确提示：
- 上传失败
- 解析失败
- 发布包格式不合法（散文件、非 zip、缺少 SKILL.md、文件数或体积超限）
- 首次发布 Slug 与已有 Skill 重复
- 路径无效
- 无权限查看
- 无权限安装
- 文件不支持预览
- skill 冲突 / 已存在
- 锁单失败
- 权限已收缩，当前版本不可更新
- 网络不可用（离线模式提示）
- 会话失效，需要重新登录
- 客户端更新检查失败
- 客户端更新包下载失败
- 客户端更新包 Hash / 签名校验失败
- 本地 Tauri Adapter 命令失败或超时
- 项目路径缺失、无效或不可写

## 12.3 成功提示
以下动作需提供明确反馈：
- 上传成功
- 发布成功
- 撤回成功
- 下架成功
- 删除成功
- 安装成功
- 启用成功
- 停用成功
- 更新成功
- 卸载成功或部分卸载成功
- 本地工具与项目扫描完成，并展示扫描目标数与需关注项数量
- 审核提交成功
- 客户端更新包下载成功
- 客户端更新程序已启动

## 12.4 关键确认弹窗
以下操作建议二次确认：
- 删除（归档）
- 下架
- 撤回
- 覆盖更新
- 卸载
- 安装客户端更新包
- 冻结用户
- 修改用户密码
- 删除部门
- 删除用户

## 12.5 登录弹窗触发规则
- 启动后默认进入主页，不以全屏登录页阻断使用
- 登录弹窗使用手机号 + 密码登录；手机号是唯一登录凭证，格式为 1 开头的 11 位数字；用户名称仅用于外显
- 登录弹窗提供“记住密码”和“自动登录”选项；自动登录依赖本机已记住密码，取消记住密码时必须同时关闭自动登录并清除本机保存的密码
- 启动时如存在有效会话 token，优先恢复会话；会话 token 缺失或失效且用户已开启自动登录时，客户端使用本机保存的手机号、密码和服务地址重新登录并完成 `/desktop/bootstrap`
- 游客点击社区作者工作区 `发布 / 我的`、通知中的远端跳转、管理入口，或触发社区中的 Star / 安装 / 更新 / 远端同步等需要后端鉴权的动作时，弹出登录框
- 登录成功后返回用户触发登录前所在的页签或动作
- 会话失效时清空远端身份，隐藏管理入口，并回到主页
- 登录态点击右上角“我的信息”时，可查看当前身份与服务状态，并支持使用“当前密码 + 新密码”修改本人密码

## 12.6 本地命令失败提示
- 安装、更新、启用、停用、卸载等本地命令必须带有动作语义，例如“安装 Skill 失败”“启用失败”“卸载失败”。
- 本地命令超时需提示用户确认 Tauri Adapter 是否仍在运行，并允许重新扫描或重试。
- 本地命令失败后必须刷新本地 Bootstrap 和扫描摘要，避免界面停留在乐观更新状态。
- 卸载成功但仍有目标目录清理失败时，需以“部分成功”提示，并列出需要手动清理的目标数量或入口。

# 13. 组织与人员变化规则

## 13.1 统一原则
用户或管理员发生变更后，**已发布的 skill 不变**。

## 13.2 后续版本 处理范围
后续版本 仅明确以下规则：
- 用户换部门后，已发布 skill 的归属信息不自动变更
- 管理员角色调整后，已发布 skill 的历史审核记录不追溯修改
- 已发布 skill 不因人员变化自动下架、删除或重新审核
- 审核链路始终按用户**当前所属部门**实时计算

## 13.3 暂不考虑事项
以下内容 后续版本 暂不考虑，进入 预留版本 方向：
- 发布者离职后的维护权自动移交
- 审核员离职后的历史展示修正
- 部门合并 / 删除后的授权范围自动迁移
- 冻结用户后已发布 skill 的托管接管规则

# 14. 非功能性体验要求

## 14.1 桌面客户端适配
- 当前版本 仅交付 Windows exe 安装包
- 后续多系统客户端扩展不进入 当前版本

## 14.2 多语言
- 页面文案支持国际化
- 默认依据地区自动识别语言
- 用户可手动切换

## 14.3 信息一致性
以下状态需全局一致：
- 已安装
- 已启用
- 有更新
- 审核中
- 已下架
- 已归档
- 权限已收缩不可更新

## 14.4 权限反馈一致性
当用户无权限时，需要统一的表现形式：
- 完全不可见
- 摘要公开
- 详情公开但不可安装
- 全员可安装

## 14.5 离线场景行为规则

> **V1.2 新增**

客户端在离线（断网）状态下的行为规则：

### 可执行操作
- 已安装 skill 正常使用
- 切换 skill 启用/停用（在本地工具/项目目录中操作 copy 副本）
- 浏览已缓存的本地数据
- 游客直接进入主页、社区、本地、设置

### 不可执行操作
- 发布 skill
- 审核 skill
- 安装/更新 skill（需服务端中转下载）
- 修改权限范围/公开级别
- 搜索社区 Skill（需服务端数据）
- 查看服务端通知或管理入口

### 恢复网络后
- 客户端仅同步离线期间发生的本地启用/停用变更
- 权限、下架、版本、可安装资格等治理状态以服务端为准
- 若离线期间服务端有版本更新，恢复后提示用户
- 若离线期间服务端发布了客户端新版本，恢复网络并验权成功后提示用户
- 若离线期间服务端权限已收缩或 skill 已下架，恢复后按权限收缩/下架规则提示，不自动删除本地已安装副本
- 若用户具备 `review` / `admin_departments` / `admin_users` / `admin_skills` 菜单权限，恢复联网并验权成功后再显示“管理”入口
