# 统一术语表

> 本文档定义系统中所有核心术语的标准含义，避免不同角色理解偏差。

## 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| Skill | Skill | 企业内部可发现、审核、安装、启用、版本化管理和权限控制的能力单元，系统最小管理对象 |
| skillID | skillID | Skill 的唯一标识符，同时也是安装后本地文件夹名称。仅允许小写字母、数字、连字符 |
| 显示名称 | Display Name | Skill 在市场中展示的友好名称，允许中文、空格、特殊字符 |
| 用户名称 | Username | 账号对外展示名称，用于社区、作者、审核提交人和管理列表，允许重名 |
| 手机号 | Phone Number | 当前自建账号体系的唯一登录凭证，格式为 1 开头的 11 位数字，不对外替代用户名称展示 |
| 市场 | Marketplace | 企业内部的 Skill 目录 + 审核中心 + 使用管理中心 + 治理平台 |

## 权限相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 授权范围 | Authorization Scope | 定义谁可以安装、下载、启用和使用该 skill。可选：本部门/本部门及下级/指定多部门/全员 |
| 公开级别 | Visibility Level | 定义非授权用户对 skill 的可见程度。三级：摘要公开/详情公开/全员可安装 |
| 摘要公开 | Summary Visible | 非授权用户可看到摘要信息，不可看详情，不可安装 |
| 详情公开 | Detail Visible | 非授权用户可查看完整详情，但不可安装/启用 |
| 全员可安装 | Public Installable | 所有用户都可查看详情并安装/启用，同时意味着全员可见详情 |
| 非公开 | Private | 仅授权范围内用户可见（默认级别） |
| 权限收缩 | Scope Narrowing | 缩小已发布 skill 的授权范围或公开级别。存量继续使用，禁止新增 |
| 权限扩大 | Scope Expansion | 扩大已发布 skill 的授权范围或公开级别。需按扩大后级别重新审核 |

## 生命周期相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 安装 | Install | Skill 从服务端下载到本地 Central Store（不等于启用） |
| 启用 | Enable | Skill 通过 symlink 优先、失败降级 copy 的方式分发到指定工具或项目的 skills 目录中生效 |
| 停用 | Disable | 从工具/项目目录中移除 skill 的副本（不影响本地 Central Store） |
| 卸载 | Uninstall | 从本地 Central Store 及所有目标目录中移除 |
| 下架 | Delist | 市场不再展示和提供安装，已安装用户不受影响（可重新上架） |
| 上架 | Relist | 下架后重新在市场展示 |
| 删除/归档 | Archive | 逻辑归档，skill 整个历史版本统一归档（终态） |
| 撤回 | Withdraw | 发布者在审核完成前撤回提交（审核中/已锁单不可撤回） |

## 审核相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 系统初审 | Automated Pre-Review | 系统自动执行的结构校验、元信息校验、安全扫描。异常时转人工，不直接拒绝；当前版本不执行风险脚本扫描 |
| 人工复核 | Manual Re-Review | 系统初审发现异常后的人工二次判断 |
| 锁单 | Claim/Lock | 审核员领取审核任务，获得排他处理权（超时 5 分钟自动释放） |
| 同级审核员 | Peer Reviewer | 同一父部门下、与发布者同级别的其他管理员 |
| 免审 | Auto-Approve | 一/二/三级管理员发布 skill 无需审核，直接发布 |
| 审核链路 | Review Chain | 从发布者到审核员的组织路径（按用户当前部门实时计算） |
| 直属部门管理员 | Direct Dept Admin | 用户当前所属部门的管理员（普通用户发布 skill 时的审核人） |

## 安装与分发相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Central Store | Central Store | 本地唯一真源，存储所有已安装 skill 原始内容（如 `~/.ai-skills/skills/`） |
| Tool Adapter | Tool Adapter | 每个 AI 工具对应的适配器，负责路径探测、格式转换、安装/卸载 |
| symlink 安装 | Symlink Install | 通过符号链接将 Central Store 中的 skill 链接到工具或项目目录；当前默认优先使用 |
| copy 安装 | Copy Install | 将转换后的 skill 文件复制到工具或项目目录；当 symlink 失败时自动降级使用 |
| Hash 校验 | Hash Verification | 比较本地文件和服务端的 Hash 值，判断是否有本地修改（全量覆盖更新） |

## 统计相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Star | Star | 收藏 + 推荐 + 搜索权重提升。实时生效数量，类似 GitHub Star |
| 下载量 | Download Count | 按用户去重的安装下载累计值。同一用户同一 Skill 多次申请安装下载凭证计一次，更新、预览和审核包下载不计入 |
| 综合排序 | Composite Ranking | 考虑搜索相关度、Star、下载量、更新时间、风险等级的综合排序 |
| 全站曝光榜 | Global Leaderboard | 面向全站的排行榜。社区侧栏当前分别展示前 10 名热榜、Star 榜和下载榜，非公开 skill 不进入 |

## 组织相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 后代部门 | Descendant Depts | 某部门下所有层级的子部门。管理员可管理所有后代部门 |
| 组织链路 | Org Chain | 从某部门到顶级部门的上级路径（用于确定审核链路） |
