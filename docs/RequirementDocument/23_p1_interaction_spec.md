## 2. 交互原则

- 真实产品前端固定为 `apps/desktop`；`ui-prototype/` 只作为原型参考，不作为联调和验收界面。
- 当前版本 基础主导航只展示使用闭环入口：首页、市场、我的 Skill、工具、项目、通知、设置。
- 在线且具备权限的管理员可在基础主导航之外追加看到审核、管理；审核页提供列表、详情、预检查结果、文件区、锁单和审核动作。
- 发布 Skill、MCP、插件不进入 当前版本 主导航；`publish` 收敛在“我的 Skill”内部标签，不新增顶级导航。
- 所有写本地文件的动作必须有明确结果反馈：安装、更新、卸载、启用、停用。
- 权限、下架、版本、可安装资格以服务端为准；客户端只做本地状态展示和启用/停用同步。
- 离线时保留本地已安装 Skill 的使用和启用/停用，不允许安装、更新、搜索市场。
- 登录、市场、通知、管理员菜单权限和连接状态必须以 live 服务端返回为准，不允许前端用假远端数据兜底。
- 任何会覆盖本地文件或删除本地副本的操作必须二次确认。

## 3. 全局布局

### 3.1 主框架

当前版本 主框架包含：

- 顶栏：应用名称、全局搜索、连接状态、通知入口、用户菜单
- 侧栏：首页、市场、我的 Skill、工具、项目、通知、设置
- 在线管理员扩展侧栏：审核、管理
- 主内容区：当前页面内容

侧栏 当前版本 不展示：

- 发布 Skill（位于“我的 Skill”内部，不单独进入侧栏）
- MCP
- 插件

### 3.2 顶栏行为

| 元素 | 交互 |
|------|------|
| 应用名称 | 点击回首页 |
| 全局搜索 | 输入关键字后跳转市场并带入 `q` 参数 |
| 连接状态 | 展示连接正常、连接中、离线、连接失败；点击可查看最近错误与重试入口 |
| 通知入口 | 展示未读角标，点击进入通知页 |
| 用户菜单 | 展示用户名、角色、部门、退出登录 |

### 3.3 连接状态文案

| 状态 | 展示文案 | 可执行动作 |
|------|----------|------------|
| connected | 已连接 | 无 |
| connecting | 正在连接 | 等待或取消重试 |
| offline | 离线模式：市场搜索、安装和更新暂不可用 | 重试连接 |
| failed | 连接失败：请检查网络或服务地址 | 重试连接、查看详情 |

## 4. 登录页

### 4.1 页面元素

- 应用名称
- 服务地址展示或选择入口
- 用户名 / 密码输入
- 登录按钮
- 连接失败提示区

### 4.2 行为

1. 用户输入账号密码。
2. 点击登录。
3. 客户端请求服务端登录或校验会话。
4. 成功后进入首页。
5. 失败时停留登录页并展示错误。

补充口径：

- 默认不预置演示账号或默认密码。
- 开发环境若需要预填测试账号，必须通过显式环境变量开启，且不进入正式验收路径。

### 4.3 错误态

| 场景 | 提示 |
|------|------|
| 账号或密码错误 | 账号或密码不正确 |
| 服务不可用 | 无法连接服务，请稍后重试 |
| 会话过期 | 登录已过期，请重新登录 |
| 网络不可用 | 当前网络不可用 |

## 6. 市场页

### 6.1 页面结构

- 搜索框
- 筛选区
- 排序区
- Skill 列表
- 简单排行榜入口或榜单区域

### 6.2 搜索

支持字段：

- Skill 名称
- 描述
- 标签
- 作者
- 部门
- skillID

行为：

1. 输入关键字。
2. 按 Enter 或点击搜索。
3. 刷新列表并保留筛选条件。
4. 搜索无结果时展示空态。

### 6.3 当前版本 必做筛选

- 部门
- 工具兼容性
- 是否已安装
- 是否已启用
- 是否仅显示我有权限查看

### 6.4 当前版本 增强筛选

- 分类
- 风险等级
- 发布时间
- 更新时间

若服务端暂未提供增强筛选字段，前端不展示对应筛选项，不展示空壳控件。

### 6.5 排序

- 综合排序
- 最新发布
- 最近更新
- 下载量
- Star 数
- 搜索相关度

### 6.6 Skill 卡片

卡片展示：

- 图标
- 显示名称
- skillID
- 简短描述
- 作者 / 作者部门
- 当前版本
- Star 数
- 下载量
- 风险等级
- 最后更新时间
- 当前状态：未安装、已安装、已启用、有更新、不可安装

卡片操作：

| 状态 | 主按钮 | 次按钮 |
|------|--------|--------|
| 可安装 | 安装 | 详情、Star |
| 已安装 | 已安装 | 详情、启用、卸载 |
| 已启用 | 已启用 | 详情、查看启用位置 |
| 有更新 | 更新 | 详情 |
| 无权限安装 | 不可安装 | 受限详情 |
| 权限已收缩 | 继续使用当前版本 | 详情 |

### 6.7 市场空态与错误态

| 场景 | 展示 | 动作 |
|------|------|------|
| 无搜索结果 | 没有找到匹配的 Skill | 清空筛选 |
| 离线 | 离线模式下无法搜索市场 | 重试连接 |
| 服务失败 | 市场数据加载失败 | 重试 |
| 无权限安装 | 你暂无权限安装该 Skill | 查看详情或返回 |

## 7. Skill 详情页

### 7.1 完整详情

展示：

- 基础信息：skillID、显示名称、描述、作者、部门、版本、更新时间
- 使用信息：适用工具、适用系统、安装说明、启用说明
- README / 使用说明
- 风险信息：风险等级、风险说明、审核摘要
- 版本信息：当前版本、最新版本、历史版本摘要
- 当前设备状态：未安装、已安装、已启用、有更新、不可更新
- 操作区

### 7.2 操作区行为

| 操作 | 条件 | 行为 |
|------|------|------|
| 安装 | `canInstall = true` 且未安装 | 获取 download-ticket，下载，校验 Hash，写入 Central Store |
| 更新 | `canUpdate = true` 且有更新 | 确认覆盖风险后下载并覆盖 |
| 启用到工具 | 已安装 | 打开目标选择弹窗 |
| 启用到项目 | 已安装 | 打开项目选择弹窗 |
| 卸载 | 已安装 | 打开卸载确认弹窗 |
| Star / 取消 Star | 可见详情或摘要 | 调用 Star 接口并更新数量 |

### 7.3 受限详情

当 `detailAccess = summary` 时展示：

- Skill 名称
- 发布人
- 部门
- Star 数
- 下载量
- 分类 / 图标
- "该 Skill 暂未向你开放详情"

当前版本 不展示：

- README
- 审核摘要
- 包下载信息
- 申请访问 / 申请使用

### 7.4 安装进度

安装进度阶段：

1. 获取下载凭证
2. 下载包
3. 校验包大小和文件数
4. 校验 Hash
5. 写入 Central Store
6. 完成

失败时必须显示失败阶段和重试入口。

## 8. 我的 Skill

### 8.1 页面结构

- 页签：已安装 / 我发布的 / 发布 Skill
- 游客或离线状态默认只展示"已安装"
- 登录后根据作者权限展示完整页签

### 8.2 已安装

页面结构：

- 搜索已安装
- 筛选：全部、已启用、有更新、权限已收缩、异常
- 已安装列表
- 批量操作：当前版本 不做

列表字段：

- Skill 名称
- 当前本地版本
- 市场最新版本
- 是否有更新
- 已启用工具
- 已启用项目
- 最近启用时间
- 启用状态
- 是否受权限收缩限制

操作：

| 操作 | 行为 |
|------|------|
| 查看详情 | 进入 Skill 详情页 |
| 更新 | 进入更新确认和下载流程 |
| 卸载 | 展示引用位置并二次确认 |
| 启用 / 停用 | 打开目标选择或执行目标停用 |
| 编辑启用范围 | 打开工具/项目目标选择弹窗 |

权限收缩提示：

当 `isScopeRestricted = true`：

- 列表展示"权限已收缩"
- 更新按钮禁用
- 新增启用范围禁用
- 文案："可继续使用当前版本，但不可更新或新增启用位置"

异常状态：

| 场景 | 展示 | 动作 |
|------|------|------|
| Central Store 缺失 | 本地副本缺失 | 重新安装 |
| 启用目标缺失 | 启用位置不可用 | 重新启用或停用 |
| 部分卸载失败 | 部分位置移除失败 | 查看失败路径、重试 |
| 本地 Hash 变化 | 本地内容已变更 | 更新时提示覆盖 |

### 8.3 我发布的

列表字段：

- Skill 名称
- skillID
- 当前版本
- 当前状态
- 授权范围
- 公开级别
- 最近更新时间
- 下载量
- Star 数

按钮规则：

| 状态 | 允许展示的动作 |
|------|----------------|
| `published` | 查看详情、下架、发布新版本、修改权限 |
| `delisted` | 查看详情、上架、删除（归档）、发布新版本、修改权限 |
| `archived` | 查看详情 |
| 待审 / 退回 / 拒绝 | 查看详情、撤回或重新提交（按服务端可执行动作集合展示） |

交互规则：

- 不同时展示互斥的上架/下架按钮
- 所有状态变更均走二次确认
- 成功后刷新列表与详情；失败时展示服务端错误

### 8.4 发布 Skill

表单字段：

- skillID
- 显示名称
- 描述
- 图标（可选）
- 标签 / 分类
- 版本号
- 变更说明
- 授权范围
- 公开级别
- 上传目录或 zip

提交行为：

- 先展示预检查结果，再允许提交
- 提交成功后进入作者提交详情页
- 发布页不提供市场展示效果预览

### 8.5 作者提交详情文件区

- 左侧展示提交包文件列表
- 右侧展示预览或下载区域
- 默认优先选中 `SKILL.md`；若不存在则选中第一个可预览文件
- `.md` / `.markdown` / `.txt` 支持在线预览
- 其他文件只显示下载入口和“不支持预览”
- 服务端返回 `truncated=true` 时，UI 显示“内容已截断”

### 8.6 审核详情文件区

- 与作者提交详情复用同一文件区布局和文件类型规则
- 审核动作区与文件区并排或上下排布，但文件区始终可见
- 审核员可在不下载 ZIP 的情况下查看 `SKILL.md`、`notes.txt` 等文本文件

## 14. 二次确认弹窗

| 操作 | 标题 | 关键文案 |
|------|------|----------|
| 覆盖更新 | 确认覆盖更新 | 本次更新将直接覆盖本地内容 |
| 卸载 | 确认卸载 Skill | 将从 Central Store 及所有启用位置移除 |
| 启用覆盖 | 确认覆盖目标目录 | 目标目录已存在，继续后会覆盖 |
| 全部已读 | 全部标记为已读 | 所有通知会被标记为已读 |

## 15. 当前版本 验收清单

- 普通用户 当前版本 导航不展示审核、管理、MCP、插件正式入口；发布入口位于“我的 Skill”内部。
- 在线且具备权限的管理员可看到审核、管理；审核页提供锁单、通过初审、同意、退回、拒绝和文件预览。
- `publish` 不进入主导航，但在“我的 Skill”中提供正式页面、上传和状态跟踪。
- 作者页根据 Skill 当前状态只展示唯一正确的上架 / 下架 / 归档动作。
- 作者提交详情和审核详情都支持 `.md` / `.markdown` / `.txt` 文件在线预览，其余文件仅下载。
- 首页只展示连接状态、安装摘要、推荐/最近更新、通知摘要和 当前版本 快捷入口。
- 市场搜索、筛选、排序、完整详情、受限详情可用。
- 无权限、离线、服务失败、无搜索结果均有明确提示。
- 安装、更新、卸载、启用、停用均有进度、结果和失败反馈。
- Tool Adapter 检测失败时可手动配置路径。
- 项目可手动添加，并能展示项目级优先规则。
- 离线时可查看本地已安装并启用/停用，不可搜索、安装、更新。
